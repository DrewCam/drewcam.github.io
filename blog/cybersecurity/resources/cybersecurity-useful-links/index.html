    Cybersecurity - useful links, Training and Certs¶

    Collection of links and resources that could be useful if you are looking to get into Cybersecurity.

    Intro to Cyber¶

    • Cisco Skills for all - Fair amount of free content, certs and badges.
    • Career path - Junior Cybersecurity Analyst
    • Rangeforce - Cybersecurity Essentials
    • Security Analyst Induction - WA Cyber Security Unit (DGOV Technical)

    Gamified/CTF¶

    Some gamified learning platforms for learning and tinkering.

    • tryhackme Has loads their new soc pathways are pretty good.
    • Kusto Detective Agency KQL is a query language used in Azure Sentinel and Defender. see Rod Trent's MustLearnKQL for more
    • hackthebox Gamified pen testing labs and learning.
    • Oh My Git! - Open source game for learning Git.

    SOC Analyst¶

    • Letsdefend.io - Soc Analyst path
    • Rangeforce - SOC Level 1

    Microsoft Tooling¶

    MS Exams can be taken online or in person. The training is free but the exams do cost money. If you keep an eye out for MS events such as Ignite, they will often have free or discounted exam vouchers.

    • MS Learn
    • John Savill's Technical Training Youtube
    • Microsoft Ninja Training
    • Microsoft Azure Fundamentals
    • Microsoft Azure Security Technologies - AZ-500
    • Microsoft Security Operations Analyst - SC-200
    • SC-200 Course
    • Rod Trent's MustLearnKQL
    • SC-200 - Charbel Nemnom
    • SC-200 - Labs

    CompTIA¶

    Fairly common to see CompTIA certs listed as a requirement on job listings.

    • Professor Messer Youtube
    • CompTIA A+ Pretty generalist and wouldnt be needed if you come from a support background.
    • ComptTIA Security+
    • CompTIA Network+

    GIAC¶

    I've listed the GIAC certs that look interesting, but there are a lot more.

    • GIAC Certified Incident Handler Certification
    • GIAC Security Operations Certified (GSOC)
    • GIAC Security Essentials (GSEC)
    • GIAC Open Source Intelligence Certification (GOSI)
